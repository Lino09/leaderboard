## Getting Started

To get a local copy up and running follow these simple example steps.

### You need NPM installed in your comnputer

### clone the repo

### Run the following commands

> cd todo-list <br>
> `npm install` <br> 
> `npm run build` <br>
> `npm run start`

### It will run in your http://localhost:8080/
